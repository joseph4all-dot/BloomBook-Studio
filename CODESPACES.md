# עבודה עם GitHub Codespaces

## פתיחה
1. היכנס למאגר:
   https://github.com/joseph4all-dot/BloomBook-Studio
2. לחץ:
   **Code → Codespaces → Create codespace on main**
3. לאחר שהסביבה נטענת, Codespaces יריץ אוטומטית:
   `npm install`

## הרצה
בטרמינל:

```bash
npm run dev
```

לאחר מכן פתח את הפורט 3000 או לחץ על הקישור ש-Codespaces מציג.

## העלאת הקבצים הראשונים
אם המאגר עדיין ריק:
1. העלה את כל תוכן התיקייה הזו למאגר.
2. ודא שהתיקיות `app`, `components`, `content`, `public`, `styles`, `.devcontainer` קיימות.
3. בצע Commit בשם:

```text
Sprint 1 foundation with Codespaces
```
